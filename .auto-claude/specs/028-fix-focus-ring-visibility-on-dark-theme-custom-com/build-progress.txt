# Build Progress: Fix Focus Ring Visibility on Dark Theme Custom Components

## Overview
Fix focus ring styles across custom components to use visible colors on dark backgrounds by changing focus:ring-offset-2 background color to match dark surface (#0a0d0a).

## Problem Analysis
The focus ring offset uses a white color by default which creates a jarring visual gap on dark theme backgrounds. Several components need updates:

1. **Button (tailadmin-ui/index.jsx)**: Uses `focus:ring-offset-2` without dark mode color
2. **Slider (ui/slider.jsx)**: References non-existent `dark-surface-bg` color
3. **Switch (ui/switch.jsx)**: References non-existent `dark-surface-bg` color
4. **Dialog (ui/dialog.jsx)**: Close button needs explicit dark mode handling
